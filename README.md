# Basic-BackEnd-with-Flask
This is a basic backend project in flask with CRUD methods in it and some authorization.

[![forthebadge made-with-python](http://ForTheBadge.com/images/badges/made-with-python.svg)](https://www.python.org/)

[![made-with-python](https://img.shields.io/badge/Made%20with-Python-1f425f.svg)](https://www.python.org/)

## Features
- Authorization (JWT)
- CRUD Operations
- User Leveling
- Password Encryption
- Login/Logout
- API

## Tech
This project uses a number of open source projects to work properly:
- [Flask] - Micro web framework written in the Python programming language.
- [Firebase Cloud Firestore] - A NoSQL database service that allows you to easily store and access application data.
- [Werkzeug] - An integral part of the Flask framework.

## Installation
### Flask
```
pip install Flask
```
### Flask JWT Extended
```
pip install Flask-JWT-Extended
```
### Firebase Admin
```
pip install firebase-admin
```
### Werkzeug
```
pip install Werkzeug
```
